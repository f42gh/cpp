# cout と コメント

```cpp
#include<bits/stdc++.h>
int main(){
    std::cout << "Hello, World!"; // 一行コメント
    /*
     * 複数行コメント
     * このように書く
     */
}
```

## cout

- `cout << 値` で標準出力に書き出す
- `<<` を連結して複数の値を一度に出力できる: `cout << x << " " << y`
- `using namespace std;` を書かない場合は `std::cout` と書く必要がある

## コメント

| 種類 | 書き方 | 用途 |
|---|---|---|
| 一行コメント | `// テキスト` | 行末まで無効化 |
| 複数行コメント | `/* テキスト */` | 範囲を指定して無効化 |
