# 算術演算子

```cpp
#include<bits/stdc++.h>
using namespace std;

int main(){
    cout << 10 + 5 << endl;  // 15  加算
    cout << 10 - 5 << endl;  // 5   減算
    cout << 10 * 5 << endl;  // 50  乗算
    cout << 10 / 2 << endl;  // 5   除算（整数同士 → 切り捨て）
    cout << 10 % 5 << endl;  // 0   剰余（余り）
}
```

## 注意点

- **整数除算の切り捨て**: `7 / 2` → `3`（小数点以下は切り捨て）
- 浮動小数点にしたい場合は `7.0 / 2` や `(double)7 / 2` とキャストする
- **剰余 `%`** は整数型のみ使用可。`10 % 3` → `1`

## 複合代入演算子

```cpp
x += 5;  // x = x + 5
x -= 5;  // x = x - 5
x *= 5;  // x = x * 5
x /= 2;  // x = x / 2
x %= 3;  // x = x % 3
```
